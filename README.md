# Todo Application

Fullstack todo application with authentication system using MERN Stack

## Demo

[![Watch the video](https://img.youtube.com/vi/lVdM9GcJU-M/maxresdefault.jpg)](https://youtu.be/lVdM9GcJU-M)

## Tech Stack Used

### Frontend

- React
- Zustand

### Backend

- NodeJS
- Express
- MongoDB
- AppWrite

## Main Features

- Login
- Offers Persistant sorage
- Todos with tasks
- Add Unlimited todos
- Sort by date
